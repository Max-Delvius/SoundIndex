# SoundIndex

SoundIndex is an open-source music library management and tagging tool.

Features
- Metadata tagging
- Genre & mood tagging
- Cover art fetching
- Lyrics downloading (.lrc)
- Large library management

Tech Stack
- PySide6 (UI)
- SQLite
- Mutagen
- MusicBrainz / Discogs / Last.fm
- LRCLIB

Status
🚧 Early development
